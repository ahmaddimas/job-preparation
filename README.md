# job-preparation

A friendly Next.js web app to help users prepare for job applications.

## Features

- Accepts a job posting URL input.
- Fetches and extracts core job information:
  - overview
  - requirements
  - benefits
- Highlights technical hard skills as an interactive checklist.
- Recommends learning resources based on detected hard skills.
- No database required.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Validation

```bash
npm run lint
npm run build
```
